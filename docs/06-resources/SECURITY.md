# Security Policy

## Supported Versions

The following table lists which versions of Muonroi Building Block currently receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.6.x   | :white_check_mark: |
| 1.5.x   | :x:                |
| < 1.5   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please email [security@muonroi.com](mailto:security@muonroi.com) with a detailed description. We aim to acknowledge reports within 2 business days and will keep you informed of our progress. When a fix is released, we will credit the reporter unless requested otherwise.
