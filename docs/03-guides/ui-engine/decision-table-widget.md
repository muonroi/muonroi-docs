# Decision Table Widget

The primary authoring surface is `mu-decision-table`.

## Important attributes

- `api-base`
- `validate-endpoint`
- `export-endpoint`
- `feel-endpoint`
- `history-endpoint`
- `reorder-endpoint`
- `table-id`
- `enable-version-diff`

Use `MuDecisionTableReact` from `@muonroi/ui-engine-react` when embedding the widget in React applications.
